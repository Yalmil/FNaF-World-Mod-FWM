## 🐛 ¿Cómo puedo contribuir?

### 💻 Enviar texturas/animaciones

Este es el proceso para contribuir:

#### Para Contribuciones Pequeñas (correcciones rápidas)
1. Haz un fork del repositorio
2. Crea una rama para tu cambio
3. Haz tus modificaciones
4. Envía un Pull Request

#### Para Contribuciones Grandes
1. **Abre un issue primero** para discutir tu propuesta
2. Espera feedback para asegurarte de que va en la dirección correcta
3. Sigue los pasos anteriores una vez validada la idea

## 🔧 Guía Técnica

### Commits

Queremos mantener un historial limpio y legible:

- Usa mensajes claros en imperativo: "Añade función X" en lugar de "Añadida función X"

### Pull Requests

Al enviar un Pull Request:

1. **Título descriptivo**: Explica qué hace el PR
2. **Descripción detallada**:
   - ¿Qué cambios introduces?
   - ¿Por qué son necesarios?
3. **Referencia el issue relacionado**: Ej: "Closes #123"
4. **Sé paciente**: Los mantenedores revisaremos tan pronto como podamos

## 🚀 Configuración del Entorno

Si quieres ejecutar el proyecto localmente:

```bash
# Clona tu fork
git clone https://github.com/TU_USUARIO/FNaF-World-Mod-FWM.git

# Entra al directorio
cd FNaF-World-Mod-FWM

# Instala dependencias (ejemplo con npm, ajusta según tu lenguaje)
npm install

# Ejecuta el proyecto
npm start
