<h2>Pasos para colaborar en FWM</h2>
<ul>
  <li>1. Para comenzar a colaborar piensa en que y como vas a ayudar con las texturas. Ejemplo: Quiero remplazar a Freddy por Aure.</li>
  <li>2. Una vez hayas pensado que vas a modificar, haz las texturas. Recuerda mantener las mismas dimensiones de la imagen original y que sea .png. Ejemplo: Ahora, descargo la foto de perfil de Aure y la redimensiono para cambiar a Freddy.</li>
  <li>3. Cuando ya tengas tú imagen, subela a Github, recuerda mantener: dimensiones exactas, formatos exactos (.png) y que tenga fondo transparente. Para subir correctamente tú imagen asegurate de meterlo en la carpeta correcta. Ejemplo: Ahora iré a la carpeta "animaciones" y subiré mis archivos con el nombre de FreddyAureSelectFrame1, FreddyAureSelectFrame2 y FreddyAureSelectFrame3</li>
</ul>
